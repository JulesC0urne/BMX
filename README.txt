Gestion de courses de BMX
##########################

Cette application est destiné à la gestion de tournois départementaux et régionaux de BMX. 
Elle permet d'enregistrer des participants, de les inscrire à des tournois. 
Chaque participants se voit un nombre de points attribuer à la fin de chaque courses. De même sa position de départ est choisi de manière équitable via un algorithme de brassage des couloirs.

Requirements
------------
Pour installer les dépendances, utiliser PIP :

pip install -r requirements.txt


Utilisation
-----------

Pour lancer le serveur en local :

python ./app.py
